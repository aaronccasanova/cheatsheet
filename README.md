## CLI

```
  $ npx chtsht --help

	Usage
	  $ npx chtsht <name>

	Options
	  --exec, -e  Execute the command

	Examples
	  # Show select list of all cheat sheets
	  $ npx chtsht

	  # Show the docker cheat sheet
	  $ npx chtsht docker

		# Execute the command
		$ npx chtsht -e docker
```
